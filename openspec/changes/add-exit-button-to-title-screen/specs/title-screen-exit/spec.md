## ADDED Requirements

### Requirement: Exit Game Button Interaction
El sistema SHALL permetre el tancament de l'aplicació mitjançant un botó dedicat ("btnExitGame") situat a la pantalla de títol.

#### Scenario: Successful application exit
- **WHEN** l'usuari fa clic al botó `btnExitGame`
- **THEN** el sistema ha d'emetre un log de confirmació i tancar el procés de l'aplicació
