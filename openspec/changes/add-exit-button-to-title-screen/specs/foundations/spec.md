## ADDED Requirements

### Requirement: Clean Application Shutdown
El sistema SHALL realitzar un tancament ordenat en rebre la instrucció de sortida des de la interfície d'usuari.

#### Scenario: Verify exit log
- **WHEN** el procés de tancament s'inicia
- **THEN** s'ha de mostrar el missatge "Sortint del joc..." a la consola per a fins de depuració
